# Production Rates

Labor production rates (units per hour/day) by trade and task. Structured rows
follow `../Data_Formats/Production_Rates_Template.csv`.

## Use
- Convert scope quantities into labor time → labor cost.
- Calibrate against actuals logged in `../Post_Mortems.md`.

## Rules
- State crew size and unit basis. Mark unverified rates `[confirm]`.
