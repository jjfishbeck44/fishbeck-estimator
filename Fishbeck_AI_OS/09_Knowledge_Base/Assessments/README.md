# Assessments

Frameworks and checklists for property condition assessments. Used by the
Property Analysis Agent and the `05_Digital_Products/Assessment_Pack`.

## Contents
- Room-by-room / system-by-system checklists
- Condition rating scale
- Red-flag list (safety, structural, water, electrical)
- Report template (findings → priority → recommended scope → cost range)

## Rules
- Prioritize safety/structural first.
- Tie recommendations to `../Pricing/` ranges.
- Flag items needing a licensed specialist or permit (`../Permitting_Matrix.md`).
