# SOPs (Standard Operating Procedures)

Repeatable procedures for how Fishbeck does the work — so quality and process stay
consistent across crews and jobs. Source material for the `05_Digital_Products/SOP_Pack`.

## Suggested SOPs
- Rental turn checklist / process
- Property assessment walkthrough process
- Jobsite setup, safety, and cleanup
- Change-order process
- Client communication cadence
- Punch-list and closeout

## Format per SOP
Purpose • When to use • Steps (grouped in phases) • Checklist • Owner
