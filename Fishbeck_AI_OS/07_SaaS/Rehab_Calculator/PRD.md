# Rehab Calculator — PRD

## Problem
Investors need a fast, credible rehab scope and budget to underwrite Twin Cities
deals.

## Solution
A tool that turns property inputs into a scoped rehab budget with cost ranges, using
Fishbeck's pricing and production-rate data.

## Users
Real estate investors (buy-and-hold, flip), wholesalers.

## Core value
Fast, defensible rehab numbers grounded in real local pricing.

## Data source
`09_Knowledge_Base/Data_Formats/` (pricing, material costs, production rates).
