# SaaS Roadmap

The future software arm of Fishbeck Innovations. Products grow out of internal tools
(the estimator, permit automation) and the Knowledge Base.

| Product | What it is | Audience | Stage |
|---------|------------|----------|-------|
| Rehab Calculator | Fast rehab scope + budget tool | Investors | Concept |
| Property Analyzer | Property condition + deal analysis | Investors/buyers | Concept |
| Investor Platform | End-to-end deal + project hub | Investors | Vision |
| Permit Automation | High-volume permit data entry | Internal → SaaS | Prototype |

## Principles
- Build from real workflows we already run.
- Reuse Knowledge Base data (`09_Knowledge_Base/Data_Formats/`).
- Start internal, then productize.
