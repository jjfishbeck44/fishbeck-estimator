# Search Intent Map

Map keywords to the right page type by intent.

## Intent types
- **Informational** — "how much does X cost", "checklist" → blog posts
- **Commercial** — "renovation company Twin Cities" → service / service-area pages
- **Transactional** — "get an estimate", "hire" → estimator / contact / landing pages
- **Navigational** — "Fishbeck Innovations" → homepage

## Mapping
| Intent | Example keyword | Page type | Location |
|--------|-----------------|-----------|----------|
| Commercial | Twin Cities home renovations | Service page | `01_Website/Services/` |
| Commercial | Saint Paul construction company | Service-area page | `01_Website/Service_Areas/` |
| Transactional | Twin Cities construction estimate | Estimator / Estimating page | `01_Website/Services/Estimating.md` |
| Informational | cost to renovate a bathroom | Blog post | `01_Website/Blog/` |

> Match content format to intent so pages rank and convert.
