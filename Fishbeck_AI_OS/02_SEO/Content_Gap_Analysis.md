# Content Gap Analysis

Topics and keywords we should cover but don't yet. Feeds the blog and content calendar.

## How to use
1. Review `Competitors.md` and `Keyword_Master.csv`.
2. List topics competitors rank for that we lack.
3. Prioritize by intent, volume, and fit.
4. Assign to `01_Website/Blog/` and `03_Content/`.
