# Landing Pages

Campaign- and offer-specific landing pages live here (one file per page).

Use landing pages for: paid ads, lead magnets, specific offers (e.g., "Free Rental
Turn Checklist"), or seasonal promotions. Each landing page should have a single
goal and a single CTA.

## Template per page
- **Meta:** title tag, meta description, primary keyword
- **Goal / CTA:** one clear action
- **Headline + subhead**
- **Offer / value proposition**
- **Proof** (testimonials, case studies)
- **Form / CTA**

Keep messaging aligned with `00_Brand/` and any tied lead magnet in
`06_Sales_Marketing/Lead_Magnets.md`.
