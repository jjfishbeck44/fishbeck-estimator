# Website Blog

Published and draft blog posts (one file per post). The blog drives SEO and
authority. Many posts come from `04_Case_Studies/` and the content pipeline in
`03_Content/Blog/`.

## File naming
`YYYY-MM-DD-slug.md` (e.g., `2026-06-21-cost-to-renovate-a-bathroom-twin-cities.md`)

## Template per post
- **Meta:** title tag, meta description, primary keyword, secondary keywords
- **H1 headline**
- **Intro** (hook + what the reader will learn)
- **Body** (H2/H3 sections)
- **Internal links** (to services, service areas, related posts — see
  `02_SEO/Internal_Linking_Strategy.md`)
- **CTA** (estimate or contact)

## Starter topic ideas
- What does it cost to renovate a bathroom in the Twin Cities?
- Rental turnover checklist for Twin Cities landlords
- How investors should scope a rehab before they buy
- Signs your property needs an assessment before purchase
