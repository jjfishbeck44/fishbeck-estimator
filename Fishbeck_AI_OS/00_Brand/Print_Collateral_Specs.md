# Print Collateral Specs

Design constraints for physical and print-ready assets — bids, one-pagers,
capability statements, flyers, yard signs, and door hangers. Keep all print output
consistent and production-ready.

## Measurement standard
- Use **1/8-inch (0.125") standard increments** for all spacing, margins, and
  layout dimensions. Round all measurements to the nearest 1/8 inch.
- Acceptable increments: 1/8, 1/4, 3/8, 1/2, 5/8, 3/4, 7/8, 1".
- Standard bleed: 1/8" (0.125"). Standard safe margin: 1/4" (0.25") from trim.

## Numbering rule
- **Maximum 3 numbered items** in any single list on client-facing collateral.
- If a process has more than 3 steps, group into a maximum of 3 phases.
- This keeps printed materials scannable and reduces visual clutter.

## Brand colors (print)
- Navy `#1B3A5C` (primary)
- Gold `#C8963E` (accent)
- Confirm CMYK equivalents with the printer before production.

## Standard sizes
- One-pager / capability statement: 8.5" × 11"
- Bid / proposal: 8.5" × 11"
- Business card: 3.5" × 2" (1/8" bleed)
- Yard sign: 24" × 18" [confirm]
- Door hanger: 4.25" × 11" [confirm]

## File output
- Print: PDF/X, 300 DPI, CMYK, fonts embedded, 1/8" bleed.
- Screen: PNG/SVG, RGB.

## Type & logo
- Use brand fonts (see `Brand_Style_Guide.md` once finalized).
- Maintain clear space around the FI logo equal to the height of the "F".

> Applies to everything generated in `06_Sales_Marketing/` intended for print.
